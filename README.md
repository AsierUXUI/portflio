# Portfolio

Personal UX/UI portfolio. React + Vite, no backend.

## Structure

- `src/data/projects.js` — case study content. Edit this to replace placeholder projects with real ones.
- `src/pages/Home.jsx` — hero, work grid, about, contact sections.
- `src/pages/CaseStudy.jsx` — case study detail page, rendered per project from `projects.js`.
- `src/components/` — Nav, Footer, ProjectCard.

## Development

```
npm install
npm run dev
```

## Build

```
npm run build
npm run preview
```
